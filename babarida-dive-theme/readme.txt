Ultra-premium luxury WordPress theme for Babarida Dive Center - professional diving, snorkeling, liveaboard, and watersport company operating in Bunaken, Siladen, Bangka, and Lembeh, North Sulawesi, Indonesia.

== Description ==

Babarida Dive Center theme is a world-class, multi-million-dollar quality marine tourism platform built natively for WordPress. It features:

* Cinematic hero with video support
* Real-time world clock system (8 timezones)
* Mega menu navigation with destination submenus
* Custom Post Types: Destinations, Trips, Liveaboards, Hotels, Testimonials, Partners, FAQ, Bookings, Courses, Water Sports
* Dynamic pricing engine with seasonal multipliers
* Complete booking system with check-in flow
* CRM with customer profiles and loyalty program
* Payment gateway integration (Midtrans, Xendit, Stripe, PayPal, Bank Transfer)
* Multi-currency support (USD, EUR, SGD, AUD, IDR)
* AI chat assistant for visitor queries
* Real-time weather & ocean conditions (OpenWeatherMap API)
* Enterprise admin dashboard with analytics, reports, and system health
* 9 custom user roles (Manager, Booking Staff, Dive Guide, Hotel Partner, etc.)
* Internal staff chat system
* Newsletter subscriber management
* Automated backup system
* Activity logging
* PWA support with service worker and manifest
* SEO optimized with Schema markup, sitemap, and robots.txt
* Google Search Console and Google Analytics integration
* Custom login page with glassmorphism design
* Security enhancements (login limiter, CSRF, XSS protection)
* WCAG accessibility compliance
* Reduced motion support
* Translation ready (.pot file included)
* WPML and Polylang compatible
* Gutenberg compatible
* No Elementor dependency

== Installation ==

1. Upload the `babarida-dive-theme` folder to `/wp-content/themes/`
2. Activate the theme through the 'Themes' menu in WordPress
3. Go to Appearance > Customize to configure hero, contact, SEO, payment, and weather settings
4. Go to Babarida DC > Dashboard to access the admin panel
5. Navigate to Settings > Permalinks and click "Save Changes" to register rewrite rules
6. Create menu items under Appearance > Menus and assign to "Primary Navigation"
7. Add destinations, trips, liveaboards, and other content through the admin panel
